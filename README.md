# Stocks ML Project

This project uses Machine Learning to predict Apple stock prices.

## Project Structure

```
Stocks_ML/
├── data/              # Data files
├── notebooks/         # Jupyter notebooks
├── src/              # Source code
├── models/           # Trained models
├── api/              # API code
├── tests/            # Tests
├── configs/          # Configuration files
├── docker/           # Docker files
└── docs/             # Documentation
```
